# CoC-6.253
Open-source server for Clash Of Clans 6.253! (Fork of Solaree)

## Requirements
**Python** (3.9 - 3.11)

## Installation
Install CoC 6.253.5 client [from here](https://clash-of-clans.uptodown.com/android/descargar/78177)


Open CoC folder and run **`pip install requirements.txt`** in your shell

Run **`python main.py`** in your shell

Enjoy the server!
